1.  First line contains N - number of nodes, E - number of edges.

<b>N E</b>

2.  The following E lines contain 3 integers describing each edge of the graph.

<b>EdgeSource EdgeDestination EdgeWeight</b>

3.  The last line contains a source and destination for which 
    the shortest path needs to be returned.

<b>Src Dest</b>


<h6>Example 1</h6>
5 6
0 1 9
0 2 1
0 3 5
0 4 3
2 1 2
2 3 1
0 3


<h6>Example 2</h6>
